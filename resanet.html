RESANET



Développement d’une application web dédiée à la gestion des réservations des repas au restaurant d’entreprise.
Cette application, baptisée RESANET, est constituée de deux modules :


Module « Usager » :


-- Réserver un repas
-- Annuler une réservation
-- Consulter ses réservations
-- Modifier son mot de passe



Module « Gestionnaire » :


-- Créer un compte de restauration
-- Bloquer et activer un compte de restauration
-- Créditer un compte de restauration
-- Réinitialiser un mot de passe
-- Consulter l’historique des réservations d’un compte de restauration
-- Consulter les réservations pour une date donnée



Contraintes non fonctionnelles :


-- Hébergement : (Serveur HTTP Apache2) / Framework flask
-- Langage de programmation : Python 2.7
-- SGBDR : MySQL 5
-- Rendu : HTML 5 / Boostrap 3
-- IDE : Geany
-- Architecture logicielle : MVC
